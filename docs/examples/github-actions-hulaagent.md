# GitHub Actions → hulaagent

Drive a remote Hula from a GitHub Actions workflow using a hulaagent cert: store the agent YAML in repo secrets, start `hula-agent` as a background job step, send HLAP commands from the workflow. Includes the full `.github/workflows/deploy.yml`.
