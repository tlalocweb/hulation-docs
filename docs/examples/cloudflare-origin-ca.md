# Behind Cloudflare (Origin CA)

Use a Cloudflare-issued Origin CA certificate so Cloudflare terminates the public TLS and Hula presents an Origin CA cert on the back-end leg. Covers cert generation, renewal, and the `ssl.cloudflare_origin_ca:` config block.
