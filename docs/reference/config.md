# config.yaml reference

Every key that `config.yaml` accepts: type, default, environment-variable override, one-paragraph description, and a minimal example. Sections mirror the YAML root keys in the order they appear in [`config/config.go`](https://github.com/tlalocweb/hulation/blob/main/config/config.go). Source of truth is the struct tags.
