# Local dev loop with hulaagent

Use `hula-agent` on a developer laptop to drive a remote staging Hula: edit / commit / push files via HLAP without ever touching `hulactl`. Covers the per-dev agent issuance, the `PUSH-FILE` flow, and the cleanup when the developer leaves the project.
