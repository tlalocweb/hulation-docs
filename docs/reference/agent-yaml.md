# Agent YAML schema

Schema reference for the YAML emitted by `hulactl create-agent` and consumed by `hula-agent`. Field-by-field: type, required, what it's used for, and what changes if you edit a field on the runner versus on the server registry. Also covers the `--offline` mode caveat.
