# Environment variables

Every environment variable Hula and `hulactl` read at startup or on operation: `HULA_*`, `HULACTL_*`, `DB_*` (for ClickHouse config), and the `${VAR}` substitutions allowed inside `config.yaml`. Includes which variables are sensitive and the recommended secret-management approach.
