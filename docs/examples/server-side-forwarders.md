# Meta CAPI + GA4 forwarding

Forward conversion events server-to-server to Meta CAPI and GA4 Measurement Protocol with no client-side beacons and no third-party cookies. Each forwarder is consent-gated by its declared `purpose`. Full config and a list of the events that get forwarded versus dropped.
