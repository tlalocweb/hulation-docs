# HLAP — Hula Local Agent Protocol

Wire-level reference for the unix-socket text protocol that local apps use to drive `hula-agent`. Covers connection model, request grammar, response shape, the streaming format for `GET-FILE` / `BUILD`, and the full verb table (`BUILD`, `STAGING-BUILD`, `PULL`, `PUSH`, `SYNC`, `COMMIT`, `STAGE`, `PUSH-FILE`, `GET-FILE`).
