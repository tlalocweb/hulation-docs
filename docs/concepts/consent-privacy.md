# Consent & privacy

Per-server `consent_mode: opt_in | opt_out | off` controls when analytics events are persisted; `Sec-GPC: 1` is honored as a binding marketing opt-out at any setting. This page covers the consent state machine, what `/v/hello` returns under each mode, and how `Hula-Consent-Required: 1` lets an embedding CMP coordinate.
