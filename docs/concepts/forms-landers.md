# Forms, landers, hooks

Forms (lead-capture endpoints) and landers (campaign landing pages) are first-class objects with versioned CRUD and Risor hooks. This page covers the data model, the captcha integrations, and how `on_new_form_submission` / `on_lander_visit` / `on_new_visitor` hooks plug into the request lifecycle.
