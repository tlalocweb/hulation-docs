# Hooks (Risor) reference

Risor scripts that fire on `on_new_form_submission`, `on_lander_visit`, and `on_new_visitor`. This page covers the globals available to each hook, the return contract, error handling, and the templating layer (Mustache) that runs over hook source before compilation.
